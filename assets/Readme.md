## Required images

* Primary app icon `assets/icon.png` 1024×1024 px without transparency  (Expo generates smaller icons)
* `assets/adaptive-icon.png` 1024×1024 px is the foreground image used for Android adaptive icons
* Splash screen for app startup `assets/splash.png` &mdash; any large rectangle, recommended 2048×2048 px, recommended: no transparency
* Browser favorites-icon `assets/favicon.png` 48×48 px
